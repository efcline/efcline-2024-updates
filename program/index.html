1:30 - 2:45pm

 

Data Curation Network

Jennifer Darragh (Duke), Joel Herndon (Duke)

●  Intro: “Why Data Curation?”

●  Overview of the DCN--past, present, future

●  Hands-on activity: curation process for tabular dataset

●  Session Abstract: Launched in May of 2016, the Data Curation Network is an ambitious, multi-organizational attempt to harness the data curation expertise of a network of data interests representing libraries, repositories, and data organizations. This presentation explores the rationale for the DCN, its current work, and future goals while sharing some strategies that the DCN employs to curate data in the network.

●  The presentation will conclude with a data curation exercise.
